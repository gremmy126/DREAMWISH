# SecondBrain README

## 목적
이 문서는 로컬 Markdown 기반 제2의 두뇌 저장소의 시작점이다.

## 핵심 내용
- 모든 지식은 사용자 컴퓨터 안의 Markdown 파일로 저장한다.
- 이 폴더는 사람이 읽을 수 있고 AI가 바로 분석할 수 있는 지식 데이터베이스다.
- 서비스나 앱이 사라져도 `SecondBrain` 폴더 자체가 지식의 원본으로 남아야 한다.

## 요약
`SecondBrain`은 Local First, Markdown First, AI First, Graph First 원칙으로 운영된다.

## 세부 내용
- `00_System`: 시스템 규칙, AI 지침, 업데이트 기록
- `01_Profile`: 사용자 프로필, 목표, 선호, 제약
- `02_Projects`: 프로젝트 인덱스와 프로젝트별 문서
- `03_Business`: 사업 아이디어, 수익모델, 마케팅, 경쟁사
- `04_Design`: 디자인 시스템, 대시보드 UI, 브랜드, 참고자료
- `05_Development`: 기술 스택, 아키텍처, API, 데이터 구조, 작업
- `06_AI_Assistant`: AI 페르소나, 에이전트, 기억, 자동화 규칙
- `07_Logs`: 일일 로그, 의사결정, 실수 기록
- `99_Archive`: 보관 문서

## AI 메모
AI는 이 문서를 저장소 루트 설명서로 보고, 다른 문서를 만들거나 수정하기 전에 [[rules]]와 [[ai-instructions]]를 우선 확인한다.

## AI가 알아야 할 규칙
이 저장소의 원본 데이터는 Markdown 파일이며, 외부 서버 저장소를 원본으로 간주하지 않는다.
