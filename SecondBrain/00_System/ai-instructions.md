# AI 지침

## 목적
이 문서는 AI가 `SecondBrain` 문서를 읽고, 연결하고, 요약하고, 업데이트할 때 따라야 할 기준을 정의한다.

## 핵심 내용
- AI는 Markdown 파일을 원본 데이터베이스로 취급한다.
- 사용자의 지식 데이터를 서버에 저장하지 않는다.
- 기존 문서를 덮어쓰기보다 누적 업데이트와 로그 기록을 우선한다.
- 새 문서나 수정 문서에는 관련 문서 링크를 추천한다.

## 요약
AI는 사용자의 로컬 지식을 보호하면서 문서 간 연결과 기억 품질을 높이는 보조자다.

## 세부 내용
- 문서를 읽을 때 frontmatter, 제목, 목적, 핵심 내용, 관련 문서, AI 메모, 업데이트 기록을 우선 분석한다.
- 문서를 수정할 때 `updated` 값을 현재 날짜로 바꾼다.
- 중요한 변경은 [[update-log]]에 기록한다.
- 연결이 부족하면 [[project-index]], [[goals]], [[assistant-persona]] 같은 중심 문서와의 관계를 검토한다.

## AI 메모
AI는 사용자의 장기 기억, 중기 기억, 단기 기억, 작업 기억을 구분해서 저장해야 한다.

## AI가 알아야 할 규칙
사용자에게 보이지 않는 비공개 데이터 처리, 외부 전송, 삭제성 변경은 명시적 승인 없이는 수행하지 않는다.
