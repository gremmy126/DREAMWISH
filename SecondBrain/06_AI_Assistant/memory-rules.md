# 기억 규칙

## 목적
이 문서는 AI가 무엇을 기억하고 무엇을 저장하지 말아야 하는지 정의한다.

## 핵심 내용
- 모든 기억은 가능한 한 `.md` 파일로 남긴다.
- 장기 기억, 중기 기억, 단기 기억, 작업 기억을 구분한다.
- 민감한 정보와 임시 정보는 무분별하게 장기 기억으로 저장하지 않는다.

## 요약
기억 규칙은 AI 메모리의 품질과 개인정보 안전을 관리한다.

## 세부 내용
### 장기 기억
- 사용자 성향
- 선호도
- 사업 목표
- 브랜드 철학
- 회사 정보

### 중기 기억
- 진행 프로젝트
- 현재 목표
- 기술 스택
- 현재 디자인
- 현재 전략

### 단기 기억
- 오늘 작업
- 이번 주 작업
- 최근 대화
- 최근 수정한 문서

### 작업 기억
- 현재 질문
- 현재 연결 중인 문서
- 현재 분석 중인 정보

## AI 메모
AI는 장기 기억으로 저장하기 전에 해당 정보가 안정적이고 사용자에게 유익한지 판단한다.

## AI가 알아야 할 규칙
저장하면 안 되는 정보는 민감정보, 검증되지 않은 추론, 일시적 감정, 사용자가 저장을 원치 않는 내용이다.
