# 목표

## 목적
이 문서는 사용자의 장기 목표, 중기 목표, 단기 목표를 구조화한다.

## 핵심 내용
- 제2의 두뇌 AI비서 구축이 현재 핵심 목표다.
- 데이터 영구 소유와 로컬 Markdown 저장 구조를 우선한다.
- 기능은 12단계로 나누어 순차적으로 개발한다.

## 요약
목표 문서는 프로젝트와 작업의 우선순위를 판단하는 기준이다.

## 세부 내용
- 장기 목표: AI가 함께 생각하는 살아있는 지식 네트워크 구축
- 중기 목표: 로컬 Markdown 저장소, 대시보드, 파서, 연결 그래프, 검색, AI 채팅 순서로 확장
- 단기 목표: `SecondBrain` 폴더 구조와 Markdown 규칙 완성
