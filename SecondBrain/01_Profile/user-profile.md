# 사용자 프로필

## 목적
이 문서는 사용자의 장기 기억과 기본 맥락을 저장한다.

## 핵심 내용
- 사용자의 정체성, 역할, 관심사, 장기 목표를 정리한다.
- AI가 사용자 맞춤 판단을 할 때 참고하는 최상위 개인 맥락이다.
- 민감한 정보는 사용자가 명시적으로 허용한 경우에만 기록한다.

## 요약
사용자 프로필은 AI가 사용자를 이해하기 위한 장기 기억의 중심 문서다.

## 세부 내용
- 이름 또는 별칭: 추후 입력
- 주요 역할: 추후 입력
- 관심 분야: AI비서, 제2의 두뇌, 로컬 지식관리
- 작업 방식: 로컬 파일 중심, Markdown 중심, 점진적 확장
- 중요 가치: 데이터 소유권, 지속성, 연결성

## AI 메모
AI는 사용자 프로필을 추론으로 채우지 않고, 사용자 발화나 문서 근거가 있는 정보만 장기 기억으로 저장한다.

## AI가 알아야 할 규칙
민감하거나 변동성이 큰 정보는 [[memory-rules]]의 저장 기준을 확인한 뒤 기록한다.

- 장기 기억으로 확정할 항목을 분류한다.
