# 아키텍처

## 목적
이 문서는 제2의 두뇌 AI비서의 전체 시스템 구조를 정의한다.

## 핵심 내용
- Markdown 파일이 원본 데이터베이스다.
- 앱은 파일을 읽고 쓰는 인터페이스와 분석 계층을 제공한다.
- 검색, 그래프, 임베딩, 캐시는 원본이 아니라 재생성 가능한 보조 구조다.

## 요약
아키텍처는 로컬 파일 원본, 파서, 인덱스, AI 인터페이스, UI 계층으로 나뉜다.

## 세부 내용
### 계층
- 파일 계층: `SecondBrain` 폴더와 `.md` 파일
- 파싱 계층: YAML frontmatter, 제목, 섹션, 내부 링크 분석
- 인덱스 계층: 검색, 링크 그래프, 태그, 업데이트 기록
- AI 계층: 요약, 연결 추천, 기억 분류, 문서 품질 검사
- UI 계층: 대시보드, 채팅, 문서 보기, 맥락 패널

## AI 메모
