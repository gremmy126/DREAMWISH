# 참고자료

## 목적
이 문서는 디자인, 제품, 구조 설계에 참고할 자료와 자산을 기록한다.

## 핵심 내용
- 현재 참고 이미지는 `public/images/dc99c2ac-f1cf-43ce-8a7f-48e3c85f64a5.png`다.
- 이 이미지는 AI비서 대시보드, 지식 네트워크 패널, 활동 요약 레이아웃의 기준으로 사용한다.
- 실제 구현 시에는 CRM과 사이드바 지식 그래프 메뉴를 제외한다.

## 요약
참고자료 문서는 디자인 의사결정의 근거 자산을 모아두는 곳이다.

## 세부 내용
### 현재 참고 이미지에서 가져올 요소
- 좌측 고정 사이드바
- 상단 검색/명령 입력
- 패널 기반 정보 구조
- AI 활동 요약
- 최근 활동
- 빠른 액션
- 채팅과 맥락 패널의 동시 배치

### 제외할 요소
- CRM 메뉴와 CRM 세부 화면
- 사이드바의 지식 그래프 메뉴
- 실제 자동화 실행 영역

## AI 메모
AI는 참고 이미지를 절대 복제 대상으로 보지 말고, 구조와 밀도, 정보 배치 원칙의 참고로 사용한다.

## AI가 알아야 할 규칙
