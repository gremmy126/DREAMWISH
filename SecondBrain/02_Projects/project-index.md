# 프로젝트 인덱스

## 목적
이 문서는 모든 프로젝트의 허브 역할을 한다.

## 핵심 내용
- 제2의 두뇌 AI비서 프로젝트를 중심 프로젝트로 관리한다.
- 12단계 개발 로드맵을 프로젝트 단위로 추적한다.
- 각 프로젝트는 목표, 디자인, 개발, 사업 문서와 연결되어야 한다.

## 요약
프로젝트 인덱스는 저장소 안의 작업과 지식을 프로젝트 중심으로 연결한다.

## 세부 내용
### 현재 프로젝트
- [[example-project]]: 제2의 두뇌 AI비서 초기 프로젝트

### 12단계 로드맵
1. 로컬 지식 폴더 구조 + Markdown 규칙 설계
2. AI비서 대시보드 디자인 이미지/와이어프레임 제작
3. 로컬 파일 읽기/쓰기 엔진 구현
4. Markdown 파서 + 메타데이터 분석
5. 노트 간 연결 그래프 구현
6. 검색/RAG/임베딩 로컬 저장 구조
7. AI 채팅 비서 연결
8. 자동 업데이트 규칙 엔진
9. 프로젝트/디자인/사업/개발별 AI 에이전트
10. Obsidian 연동/플러그인 구조
11. 백업/동기화/내보내기
12. 배포용 데스크톱 앱 완성

## AI 메모
AI는 새 작업을 만들 때 반드시 어떤 프로젝트와 연결되는지 확인한다.

## AI가 알아야 할 규칙
프로젝트 문서는 사업, 디자인, 개발, 로그 문서와 최소 1개 이상 연결되어야 한다.
