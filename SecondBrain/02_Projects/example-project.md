# 제2의 두뇌 AI비서 프로젝트

## 목적
이 문서는 로컬 Markdown 기반 제2의 두뇌 AI비서 프로젝트의 핵심 방향을 정리한다.

## 핵심 내용
- 사용자의 컴퓨터 안에 있는 `.md` 파일을 진짜 데이터베이스로 사용한다.
- AI비서 앱은 이 파일들을 읽고, 연결하고, 요약하고, 실행하는 인터페이스다.
- 현재는 1단계 저장소 구조와 문서 규칙을 완성한다.

## 요약
이 프로젝트는 AI가 사용자와 함께 기억하고 추론하는 로컬 퍼스트 지식 네트워크를 만든다.

## 세부 내용
- 데이터 원본: `SecondBrain` 폴더의 Markdown 파일
- 디자인 기준: 참고 이미지 기반의 업무형 대시보드
- 기능 제외: 현재 단계에서는 실제 AI 채팅, 검색, 임베딩, DB, 자동화, CRM 미구현
- 핵심 연결: 사용자 목표, 개발 아키텍처, 디자인 시스템, AI 기억 규칙
